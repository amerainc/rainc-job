# rainc-job

> 毕业设计,一个分布式任务调度平台。
>
> 参考了[徐雪里大神的xxl-job](https://github.com/xuxueli/xxl-job/)

## 技术栈
- spring-boot
- spring-data-jpa
- spring-webflux
- Feign
- Netty
- lombok
- Mysql
## 开发环境
- jdk1.8
- mysql8.0.16
- maven3.6.2
